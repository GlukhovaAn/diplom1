# Дипломная работа

Проектирование и разработка WEB-интерфейса конструктора подбора стильных образов, капсул одежды, гардероба для интернет-магазина одежды

# Стек

- Node.js, Express.js
- React, Vite
- MongoDB

# Сервер

модуль аутентификации пользователей:
/api/auth
модуль работы с товарами:
/api/products
модуль корзины:
/api/cart
модуль заказов:
/api/orders
модуль рекомендаций:
/api/recommendations
модуль пользователей:
/api/users
модуль капсульных гардеробов:
/api/capsules

/api/admin/products

# Клиент

Пользовательская часть включает страницы, обеспечивающие основной функционал интернет-магазина и конструктора образов:
главная страница:
/home
Отображает основную информацию, рекомендации и популярные товары.
каталог и карточка товара:
/product
Позволяет пользователю просматривать информацию о товаре, выбирать параметры и добавлять его в корзину или в конструктор образов.
корзина:
/cart
Отображает выбранные товары, позволяет изменять их количество и удалять позиции.
оформление заказа:
/checkout
Реализует ввод данных доставки и подтверждение заказа.
заказы пользователя:
/orders
Предоставляет доступ к истории заказов и их статусам.
личный кабинет:
/profile
Содержит информацию о пользователе и его настройках.
избранное:
/wishlist
Позволяет сохранять интересующие товары для последующего просмотра.
капсулы:
/capsules
Показывает готовые капсулы одежды

Административный интерфейс предназначен для управления системой и доступен только авторизованным администраторам.
Структура административной панели включает следующие страницы:
вход администратора:
/admin/login
панель управления:
/admin/dashboard
Отображает сводную информацию о системе (заказы, пользователи, статистика).
управление товарами:
/admin/products
Позволяет добавлять, редактировать и удалять товары.
управление заказами:
/admin/orders
Обеспечивает контроль и обработку заказов пользователей.
управление пользователями:
/admin/customers
Предоставляет доступ к информации о пользователях.
управление капсульными гардеробами:
/admin/capsules
Позволяет создавать и редактировать капсульные образы.
